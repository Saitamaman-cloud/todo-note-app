# 今日メモTodo

個人用の「ToDo + 複数ページメモ」PWAです。スマホで今日のToDoを追加し、状態を変え、自由メモを複数ページ保存できます。データはログインなしで端末内のIndexedDBに保存します。

## ファイル構成

```text
todo-note-app/
├─ index.html
├─ style.css
├─ app.js
├─ db.js
├─ manifest.json
├─ service-worker.js
├─ README.md
└─ icons/
   ├─ icon-192.png
   └─ icon-512.png
```

## ローカル起動方法

1. ターミナルで `todo-note-app` フォルダへ移動します。
2. 次のコマンドを実行します。

```bash
python -m http.server 8000
```

3. ブラウザで `http://localhost:8000/` を開きます。

PWAとService Workerは `file://` では正しく動かないため、ローカルサーバー経由で確認してください。

## スマホで使う方法

同じWi-Fi内のスマホからPCのIPアドレスを使って開きます。

```text
http://PCのIPアドレス:8000/
```

Android Chromeでの利用を主対象にしています。iPhone Safariでもホーム画面追加と基本利用ができる構成です。

## ホーム画面に追加する方法

### Android Chrome

1. Chromeでアプリを開きます。
2. メニューから「ホーム画面に追加」または「アプリをインストール」を選びます。
3. 追加後、ホーム画面のアイコンから起動します。

### iPhone Safari

1. Safariでアプリを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。

## バックアップ方法

設定画面の「バックアップ出力」を押すと、ToDoとメモの全データをJSONファイルとして保存します。

ファイル名は次の形式です。

```text
today-memo-todo-backup-YYYYMMDD.json
```

## 復元方法

1. 設定画面の「バックアップ復元」を押します。
2. 保存済みのJSONバックアップファイルを選びます。
3. 確認ダイアログで同意すると、既存データを全削除してバックアップ内容に置き換えます。

## 注意事項

- データは使用中のブラウザ内に保存されます。
- ブラウザデータやサイトデータを削除すると、保存内容が消える可能性があります。
- クラウド同期、ログイン、画像添付、通知、共有機能は初期版にはありません。
- 大切なメモは定期的にバックアップしてください。

## 動作確認項目

- ホーム画面に今日の日付が表示される。
- 今日のToDo件数が、未着手・対応中・完了ごとに表示される。
- ToDo画面で今日の日付が初期表示される。
- ToDoを追加できる。
- ToDoの状態を「未着手 → 対応中 → 完了」に変更できる。
- ToDoの状態を戻せる。
- ToDoを削除できる。
- 日付を変えると、その日付のToDoだけが表示される。
- メモ一覧で新規メモを作成できる。
- メモ編集画面でタイトルと本文を保存できる。
- 既存メモを開いて編集できる。
- メモを削除できる。
- タイトルと本文を対象にメモ検索できる。
- バックアップJSONを出力できる。
- バックアップJSONから復元できる。
- `manifest.json` が読み込まれる。
- `service-worker.js` が登録され、主要ファイルがキャッシュされる。
- オフライン状態でもアプリ画面が起動する。
